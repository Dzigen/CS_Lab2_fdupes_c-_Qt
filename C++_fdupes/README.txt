Программа реализует упрощенные функции  утилиты fdupes.

Программа сканирует заданный параметром командной строки каталог и его подкаталоги,
для каждого обнаруженного набора идентичных копий предлагает пользователю выбор,
какую из копий сохранить (или пропустить, т.е. сохранить все), и удалить ненужные копии.

dir_0 - тестовый каталог для проверки работы программы

3(размер(в байтах))
  294(хэш)
    .../dir_0/text_dup11
    .../dir_0/text_dup12
    .../dir_0/text_dup14
    .../dir_0/text_ndup13
4
  394
    .../dir_0/dir_1/text
  435
    .../dir_0/dir_1/dir_2/text_ndup31
    .../dir_0/dir_1/dir_2/text_ndup32
8
  860
   .../dir_0/dir_1/text_dup22
   .../dir_0/text_dup21

для запуска программы нужно выполнить команды:
	make(создать испольняемый файл)
	./fdupes dir_0
